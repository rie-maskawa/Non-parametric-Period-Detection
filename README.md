# Non-parametric-Period-Detection

This repository provides the code accompanying the manuscript:

**"Ecosystem synchronization and functional dynamics in the gut microbiome captured through automated high-resolution sampling" (Maskawa, Kurokawa et al., 2025)**  

We provide reproducible analysis pipelines for:

- Phase reconstruction using NPPD  
- Hierarchical clustering of genome-resolved phases  
- Computation of the Kuramoto order parameter
